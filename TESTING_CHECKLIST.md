## 8. Безопасность

- [ ] `POST /api/listings` без `X-Telegram-Init-Data` → `401`
- [ ] `POST /api/listings` с чужим `initData` → `403`
- [ ] `PATCH /api/listings/{id}` без авторизации → `401/403`
- [ ] `DELETE /api/listings/{id}` без авторизации → `401/403`
- [ ] `GET /api/admin/listings` без авторизации → `401/403`
- [ ] Загрузка файла с неверным MIME (не изображение) → отклоняется бэкендом

- [ ] Фильтр с несуществующими параметрами — возвращает пустую ленту без ошибки
